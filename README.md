# ET-RPG-DEMO
仅供学习交流.  
目标是在ET框架的基础上,做一个支持多人对战的ARPG的DEMO.   
借以学习服务器端开发.和了解服务器/客户端交互细节.  

## 项目特点   
1.同步方式为状态同步(似乎有点像混合同步? 反正刚接触服务器开发,不太了解这块的区别)   
2.双端物理(BOX2D,模拟3D)  
3.ARPG,支持多人对战

# 目前目标和进度的说明
